# Hyper Markdown Book (HM Book)

Приложение, разработанное с целью предоставить удобный графический интерфейс для работы с коллекцией данных в формате Markdown, с возможностью редактирования файлов в режиме WIYSWIYG. Главной отличительной чертой приложения HM Book от аналогичных является то, что оно позволяет работать "напрямую" с файлами, не искажая их формат, не внося никаких служебных измений или дополнений в файлы или в папку, в которой хранится весь набор. Для управления настройками приложение используюет только файлы конфигурации, располагаемые в отдельном пользовательском каталоге настроек.

MDBook написан на языке программирования C++. Для удобства работы использует библиотеки cmark и wxWidgets.
